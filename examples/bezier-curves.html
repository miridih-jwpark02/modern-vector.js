베지어 곡선 예제

Modern Vector.js를 사용하여 2차와 3차 베지어 곡선을 생성하고 조작하는 방법을 보여주는 인터랙티브 예제입니다.

곡선 타입

2차 베지어 곡선은 하나의 제어점을 사용하여 부드러운 곡선을 생성합니다.

곡선 속성

2px

코드 예제

// 2차 베지어 곡선 생성 예제
const shapePlugin = engine.getPlugin('shape');
const path = shapePlugin.createShape('path', {
  points: [],
  style: {
    strokeColor: '#1d4ed8',
    strokeWidth: 2
  }
});

// 시작점 추가
path.addPoint(100, 250, 'move');

// 2차 베지어 곡선 추가
path.addQuadraticCurve(
  400, 50,
  700, 250
);

// SVG path 문자열로 변환
const svgPath = path.toSVGPath();
// 결과: "M100,250 Q400,50 700,250"

베지어 곡선 사용하기

베지어 곡선은 부드러운 곡선을 생성하는 데 사용되는 매개변수 곡선입니다. Modern Vector.js는 2차와 3차 베지어 곡선을 지원하며, 이를 통해 복잡한 형태의 경로를 만들 수 있습니다.

위 인터랙티브 데모에서는 다음과 같은 기능을 사용할 수 있습니다:

  • 2차 또는 3차 베지어 곡선 선택
  • 제어점을 드래그하여 곡선 모양 조정
  • 선 색상 및 두께 설정
  • 실시간으로 생성되는 코드 예제 확인

베지어 곡선은 SVG path 문자열로도 표현할 수 있으며, Modern Vector.js는 SVG path 문자열과의 상호 변환을 지원합니다. 자세한 내용은 문서를 참조하세요.
