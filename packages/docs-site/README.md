# Modern Vector.js 문서 사이트

Modern Vector.js 라이브러리의 인터랙티브 예제와 문서를 제공하는 웹사이트입니다.

## 기능

- 라이브러리 소개 및 주요 기능 설명
- 인터랙티브 예제 (기본 도형, 변환, 패스 연산, SVG 렌더러)
- API 문서 링크

## 개발 환경 설정

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build

# 빌드 결과물 미리보기
npm run preview

# GitHub Pages 배포
npm run deploy
```

## 기술 스택

- Vite
- TypeScript
- Modern Vector.js
