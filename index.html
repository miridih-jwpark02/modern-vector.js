Modern Vector.js는 플러그인 기반의 벡터 그래픽 라이브러리입니다. Canvas와 SVG 렌더링을 지원하며, 확장 가능한 아키텍처를 제공합니다.

주요 기능

  • 플러그인 기반 아키텍처
  • Canvas 및 SVG 렌더링 지원
  • 다양한 Shape 지원 (Rectangle, Circle, Line, Path, Text)
  • 변환 (Translation, Rotation, Scale) 지원
  • Path 연산 (Union, Intersect, Subtract, XOR) 지원
새로운 기능!

이제 인터랙티브 예제를 통해 Modern Vector.js의 기능을 직접 체험해볼 수 있습니다.

사용 방법

Modern Vector.js를 사용하여 간단한 벡터 그래픽을 그리는 방법입니다:

Canvas 렌더러 사용

// 엔진 초기화
import { VectorEngine } from 'modern-vector.js';
import { ShapePlugin } from 'modern-vector.js/plugins/shapes';
import { CanvasRenderer } from 'modern-vector.js/plugins/renderers/canvas';

// 엔진 생성
const engine = new VectorEngine();

// 플러그인 등록
engine.use(new ShapePlugin());

// 렌더러 등록
const renderer = new CanvasRenderer({
  context: {
    canvas: document.getElementById('canvas')
  }
});
engine.renderer.register(renderer);
engine.renderer.setActive('canvas');

// 장면 생성
const scene = engine.scene.create();

// Shape 생성
const shapePlugin = engine.getPlugin('shape');
const rectangle = shapePlugin.createShape('rectangle', {
  x: 50,
  y: 50,
  width: 100,
  height: 50,
  style: {
    fillColor: 'red',
    strokeColor: 'black',
    strokeWidth: 2
  }
});

// 장면에 Shape 추가
scene.root.appendChild(rectangle);

// 렌더링
engine.renderer.render(scene);

SVG 렌더러 사용

// 엔진 초기화
import { VectorEngine } from 'modern-vector.js';
import { ShapePlugin } from 'modern-vector.js/plugins/shapes';
import { SVGRenderer } from 'modern-vector.js/plugins/renderers/svg';

// 엔진 생성
const engine = new VectorEngine();

// 플러그인 등록
engine.use(new ShapePlugin());

// SVG 렌더러 등록
const renderer = new SVGRenderer({
  context: {
    svg: document.getElementById('svg-container')
  },
  width: 500,
  height: 300,
  viewBox: { x: 0, y: 0, width: 500, height: 300 }
});
engine.renderer.register(renderer);
engine.renderer.setActive('svg');

// 장면 생성
const scene = engine.scene.create();

// Circle 생성
const shapePlugin = engine.getPlugin('shape');
const circle = shapePlugin.createShape('circle', {
  centerX: 150,
  centerY: 150,
  radius: 50,
  style: {
    fillColor: 'blue',
    strokeColor: 'black',
    strokeWidth: 2
  }
});

// 장면에 Shape 추가
scene.root.appendChild(circle);

// 렌더링
engine.renderer.render(scene);
참고

Modern Vector.js는 플러그인 기반 아키텍처를 사용하므로, 필요한 기능을 플러그인으로 추가하여 확장할 수 있습니다. 위 예제에서는 기본 Shape 플러그인과 렌더러 플러그인만 사용했지만, 필요에 따라 다양한 플러그인을 추가할 수 있습니다.

플러그인 개발 가이드

Modern Vector.js는 플러그인 기반 아키텍처를 사용하여 확장성을 제공합니다. 다음은 간단한 플러그인을 개발하는 방법입니다:

import { Plugin, VectorEngine } from 'modern-vector.js';

/**
 * 간단한 로깅 플러그인 예제
 */
export class LoggerPlugin implements Plugin {
  readonly id = 'logger';
  readonly version = '1.0.0';
  readonly dependencies = []; // 의존성이 없는 경우

  /**
   * 플러그인 설치
   */
  install(engine: VectorEngine): void {
    // 렌더링 이벤트 리스너 등록
    engine.events.on('render', (scene) => {
      console.log(`Scene rendered: ${scene.id}`);
    });

    // 장면 생성 이벤트 리스너 등록
    engine.events.on('sceneCreated', (scene) => {
      console.log(`New scene created: ${scene.id}`);
    });

    console.log('Logger plugin installed');
  }

  /**
   * 플러그인 제거
   */
  uninstall(engine: VectorEngine): void {
    // 이벤트 리스너 제거 등 정리 작업
    console.log('Logger plugin uninstalled');
  }
}

플러그인 사용 방법:

import { VectorEngine } from 'modern-vector.js';
import { LoggerPlugin } from './plugins/logger';

// 엔진 생성
const engine = new VectorEngine();

// 플러그인 등록
engine.use(new LoggerPlugin());
플러그인 개발 팁
  • 플러그인은 고유한 ID를 가져야 합니다.
  • 의존성이 있는 경우 dependencies 배열에 명시해야 합니다.
  • install 메서드에서 필요한 초기화 작업을 수행합니다.
  • uninstall 메서드에서 리소스 정리 작업을 수행합니다.
  • 다른 플러그인과의 통신은 이벤트 시스템을 활용하는 것이 좋습니다.
