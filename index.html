Modern Vector.js 소개

Modern Vector.js는 플러그인 기반의 벡터 그래픽 라이브러리입니다. Canvas와 SVG 렌더링을 지원하며, 확장 가능한 아키텍처를 제공합니다.

플러그인 기반 아키텍처

API 문서

Modern Vector.js의 API 문서는 여기에서 확인할 수 있습니다.

주요 클래스

  • VectorEngine - 벡터 그래픽 엔진의 메인 클래스
  • RendererService - 렌더링 관련 기능 제공
  • SceneService - 장면 관리 기능 제공
  • EventService - 이벤트 처리 기능 제공
